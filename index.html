02
🐍 Python Programming

The first part of the session focused on building a strong foundation in Python programming. Python is one of the most widely used programming languages today, known for its simplicity and readability. The following topics were covered in detail:

🧠 Logic
The concept of logic in programming was introduced — the ability to think through a problem in a clear, structured way before writing any code, covering conditions, decisions, and program flow.
💻 Programming
An introduction to programming was covered, explaining the process of writing instructions a computer can understand and execute, along with the basic structure of a Python program.
📋 Algorithm
Algorithms were explained as well-defined, step-by-step instructions for solving a problem. Planning algorithms on paper or in pseudocode before coding was emphasised as a key practice.
📦 Variables
Variables were introduced as named containers for storing data. Topics included naming conventions, value assignment, and how variables hold different data types throughout program execution.
🔢 Data Types
The session covered integers, floats, strings, and booleans. Python's automatic type identification was explored, along with how data type choice affects storage, processing, and display.
⚙️ Programs
Several basic programs were written and executed, including programs involving user input, data processing using variables and data types, and displaying output in Google Colab.
☁️ Google Colab — Hands-on Practice: Google Colab was used as the primary environment for writing and running Python code during the session. It is a free, cloud-based platform that runs in the browser with no local installation required. Notebooks were created, code was written in cells, and outputs were viewed instantly — bridging theory directly into working code.
🔁 For Loop
The for loop was introduced as a control structure used to repeat a block of code a specific number of times or over a sequence. The syntax, the range() function, and the loop variable were covered with practical executed examples demonstrating how for loops simplify repetitive tasks.
🔄 While Loop
The while loop was explained as a loop that continues executing as long as a specified condition remains true. Writing while loops, defining conditions, and updating them to avoid infinite loops were covered, followed by a comparison between for and while loops to clarify when each is appropriate.
03
🤖 Artificial Intelligence (AI)

The second part of the session introduced the field of Artificial Intelligence. AI is one of the most rapidly growing areas in technology today, and this session provided a solid foundation to understand what it is and how it is applied.

💡 What is Artificial Intelligence
AI was defined as the simulation of human intelligence by machines — enabling them to understand language, recognise patterns, make decisions, and solve problems. It is a broad field bringing together various techniques to make machines smarter.
🗂️ Types of AI
Narrow AI performs specific tasks (e.g. facial recognition). General AI can perform any intellectual task a human can. Super AI is a hypothetical future AI surpassing human intelligence in all areas.
⚖️ AI vs Machine Learning vs Deep Learning
A clear comparison was made between all three concepts to show how they relate to each other.
Concept Definition Relationship
Artificial Intelligence Any technique enabling machines to mimic human intelligence — the broadest concept. Parent Field
Machine Learning Machines trained on data to improve performance over time without explicit programming. Subset of AI
Deep Learning Uses multi-layered artificial neural networks to process and learn from large data sets. Subset of ML
